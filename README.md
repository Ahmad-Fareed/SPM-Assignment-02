# SPM-Assignment-02# SPM Assignment Project

## Project Overview
This repository contains the project developed as part of the **Fundamentals of Software Project Management (SPM)** assignment.  
The project demonstrates Agile workflow using **Jira** integrated with **GitHub** for issue tracking, sprint planning, and development management.

## Objectives
- Create and manage Epics and backlog items in Jira.
- Plan and execute sprints.
- Integrate Jira with GitHub.
- Track development using branches, commits, and pull requests linked to Jira issues.

## Project Features
- User Authentication module
- Dashboard & User Management

## Tools & Technologies
- Jira Software
- GitHub
- Agile Scrum Methodology
- Git & Version Control

## Workflow Used
1. Create Epics and backlog items in Jira.
2. Plan sprint and assign tasks.
3. Create branches using Jira issue keys.
4. Commit code linked to Jira issues.
5. Create pull requests and merge after review.
6. Track progress via Jira development panel.
